🚀 总结与进阶

Three.js 的核心流程：

  1. 初始化环境（三大件）。
  2. 创建资产（几何体 + 贴图）。
  3. 布局空间（光照 + 随机分布）。
  4. 联动交互（将 DOM 滚动与 3D 变换绑定）。
核心组件 功能描述
TextureLoader 用于给物体穿上“外衣”（如月球表面纹理）
OrbitControls 调试利器，允许鼠标拖拽缩放查看场景
MathUtils 快速处理 3D 空间内的随机数学运算

建议下一步尝试：

  • 添加 GSAP 实现更平滑的补间动画。
  • 学习 Shaders (GLSL) 编写自定义的视觉效果。
